# 桌宠素材说明

这个文件夹用于存放桌面宠物的动画图片。

## 需要准备的素材

请使用豆包、即梦、Midjourney等AI工具生成以下GIF图片：

### 1. idle.gif - 待机动画
- 宠物静止或轻微晃动的动画
- 建议尺寸：96x96 或 128x128 像素
- 建议：小猫坐着眨眼、小幅度呼吸动画

### 2. happy.gif - 开心动画
- 宠物开心的动画
- 建议：小猫跳跃、摇尾巴、眯眼笑

### 3. eating.gif - 吃东西动画
- 宠物吃东西的动画
- 建议：小猫低头吃东西、嚼东西

### 4. sleeping.gif - 睡觉动画
- 宠物睡觉的动画
- 建议：小猫趴着睡觉、有Z字符号

### 5. walking.gif - 行走动画
- 宠物行走的动画
- 建议：小猫走路、小跑

## 图片要求

1. **格式**：GIF（支持动画）或 PNG（静态）
2. **背景**：透明背景最佳
3. **尺寸**：建议 96x96 或 128x128 像素
4. **风格**：统一风格，建议可爱/治愈系

## 使用方法

1. 将生成的图片保存到此文件夹
2. 文件名必须与上述对应
3. 刷新页面即可看到效果
